# Contributing

Thanks for helping improve JSON Tree Viewer.

## Project Setup

No install or build step is needed.

1. Fork and clone the repo.
2. Open `index.html` in a browser.
3. Edit these files as needed:
   - `index.html` for app structure
   - `style.css` for layout, themes, and responsive styles
   - `script.js` for JSON parsing, tree view, graph view, zoom, pan, and file upload

## Before You Submit

Please test your change manually:

- Paste valid JSON and render it.
- Try invalid JSON and confirm the error is clear.
- Upload a `.json` file.
- Check List View and Graph View.
- Test expand, collapse, zoom, pan, and theme toggle.
- Resize the browser and check mobile layout.

## Pull Requests

- Keep changes focused.
- Match the existing HTML, CSS, and vanilla JavaScript style.
- Do not add dependencies unless there is a strong reason.
- Open an issue first for large UI or behavior changes.
- Add screenshots or a short note for visual changes.

## Good First Issues

Small fixes are welcome: UI polish, accessibility, browser quirks, examples, docs, and clearer error handling.
